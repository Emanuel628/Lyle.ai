# Lyle.ai Design Guide

## Visual Direction
Lyle should look:
- simple
- clean
- modern

## Design Tone
The design should feel:
- polished
- calm
- professional
- capable
- easy to trust

## Layout Direction
The interface should:
- use clear spacing
- avoid clutter
- feel easy to scan
- remain visually calm
- support both general users and coding-heavy users

## Visual Style Rules
- keep the layout clean
- avoid busy backgrounds
- avoid flashy effects
- avoid overdesigned futuristic styling
- keep the interface readable and structured
- use visual restraint

## Silver Influence
Silver can be used as a restrained influence, not as a loud theme.

If used, silver should feel:
- subtle
- modern
- clean
- polished

It should not feel:
- chrome-heavy
- shiny for no reason
- gimmicky
- overdone

## Design Priorities
- clarity first
- simplicity first
- modern presentation
- premium restraint
- trust-building visuals

## User Experience Through Design
The design should help users feel:
- comfortable
- confident
- not overwhelmed
- not embarrassed to ask for help

## Core Design Rule
Simple, clean, modern.
