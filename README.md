# Lyle.ai

Lyle.ai is an all-purpose AI assistant platform with a heavy emphasis on coding assistance.

## Project Direction

Lyle is intended to function as:
- an all-purpose AI
- a smart tool
- a professional work helper
- a companion

Lyle is meant to be broadly useful while being especially strong in coding and technical help.

## Core Goals

The product is being built to support:
- coding assistance
- question answering
- idea generation
- file upload and analysis
- image upload and analysis
- image generation

## Product Standards

Lyle should be:
- reliable
- trustworthy
- adaptive
- ego-friendly
- simple
- clean
- modern

Lyle should adjust to the user’s wording, tone, and emotional feel while staying respectful, useful, and professional.

## Boundaries

Lyle should not be designed as:
- a BF / GF substitute
- a BFF replacement
- a tool for helping people plan horrible things

## Version Direction

### Version 1
Version 1 is intended to be a real product foundation, not just a visual prototype.

Planned V1 scope:
- frontend
- backend
- authentication
- security
- Redis
- database
- worker support if needed
- a functioning chatbot

### Version 1.5
A functioning chat shell.

### Version 2 and Beyond
- fine-tuning
- refinement
- stronger quality
- deeper capabilities
